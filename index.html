DNS Blocker

This is a home-made ad blocker which uses a raspberry pi as the server (pi-hole FTW).

The adblockers on browsers are fine but don't work for mobile applications, I made this for my home network when I noticed my little cousin would constantly exit candy crush because she clicked an ad. So after getting tired of constantly having to exit the 10 ads she clicked every 5 minutes, I decided to solve her problem :)

I am not a hardware engineer, so tinkering with the pi was intimidating but was extremely fun and inspired me to make a magic mirror. (in progress)

Requirements/Tools:
- SD card (for Linux OS)
- Pi-Hole Software
- Access to network/router
- Programming/Linux directory knowledge
- Raspberry Pi
